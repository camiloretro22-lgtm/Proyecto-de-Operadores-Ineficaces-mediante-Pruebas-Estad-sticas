# Proyecto-de-Operadores-Ineficaces-mediante-Pruebas-Estad-sticas

Este proyecto tiene como objetivo analizar los resultados de un experimento A/B para determinar si una nueva versión de un producto o interfaz genera una mejora significativa en la tasa de conversión de los usuarios.

Se aplican técnicas de análisis de datos y pruebas estadísticas para evaluar si las diferencias observadas entre los grupos son estadísticamente significativas.
